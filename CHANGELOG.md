# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- New `Mtk` (Math) functions: `sin`, `cos`, `tan`, `asin`, `acos`, `atan`, `atan2`, `log`, `log2`, `log10`, `exp`, `cbrt`, `clz32`, `cosh`, `expm1`, `fround`, `hypot`, `imul`, `log1p`, `sign`, `sinh`, `tanh`, `trunc`.
- New instance creation syntax: `ClassName anyar()`.
- Expanded `cetakno` (console) methods: `peringatan`, `kesalahan`, `info`, `debug`, `tabel`, `hitung`, `waktu`, `akhirWaktu`, `grup`, `akhirGrup`.
- New global functions for timers: `setWaktuInterval`, `hapusWaktuInterval`, `setWaktuTunda`, `hapusWaktuTunda`.
- Static methods for `Obyek` (Object) and `Daftar` (Array): `Obyek.iku`, `Obyek.wenehno`, `Obyek.kunci`, `Obyek.nilai`, `Obyek.entri`, `Daftar.ikiDaftar`.
